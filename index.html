What does this mean?

(Check the JS console for figures)

The color of the background represents the current TERN/USD price, based on previous data from the last selected time period. Green means the current value is higher than average, red means the value is lower. The intention is to show the best spending and depositing time for BLOCKCARD accounts. Green is a good spending time, red is a good time to deposit funds.
